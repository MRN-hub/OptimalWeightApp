## Описание приложения

WPF-приложение **«Вычисление оптимального веса человека»** реализует расчёт оптимального веса на основе роста и пола пользователя, а также выводит сравнительную характеристику фактического веса.

### Формулы расчёта

| Пол | Формула |
|---|---|
| Мужчина | `(рост − 100) × 0.87` |
| Женщина | `(рост − 100) × 0.90` |

### Диапазоны допустимого ввода

- Рост: **130 – 220 см**
- Вес: **40 – 170 кг**

### Сравнительная характеристика

| Условие | Характеристика |
|---|---|
| Вес < оптимальный − 3 кг | Ниже нормы |
| оптимальный − 3 ≤ вес ≤ оптимальный + 3 | Норма |
| Вес > оптимальный + 3 кг | Выше нормы |
